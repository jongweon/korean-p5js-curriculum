# Korean Culture × p5.js

**단청 · 한글 · K-POP** — 외국인 학생을 위한 12시간 코딩 커리큘럼

한국 문화 예술(단청, 한글, K-POP)을 소재로 프로그래밍 기초와 AI 활용(바이브 코딩)을 가르치는 교육 자료입니다. 사전 코딩 경험이 없는 학생도 따라올 수 있도록 설계되었습니다.

---

## 🎯 프로젝트 소개

이 커리큘럼은 매 차시 **개념 → 한국 문화 → 실습 → AI** 네 단계를 따릅니다. 학생들은 p5.js(JavaScript 크리에이티브 코딩 라이브러리)와 AI 어시스턴트를 활용하여 인터랙티브 시각 프로그램을 직접 만듭니다.

| 항목 | 내용 |
|------|------|
| 총 차시 | 12개 레슨 (또는 6회 × 100분) |
| 총 시간 | 10~12시간 |
| 대상 | 코딩 경험이 없는 외국인 학생 |
| 도구 | p5.js Web Editor · AI assistant · 웹 브라우저 |
| 언어 | 영어 (수업 자료) |

## 📖 커리큘럼 구조

### Phase 1 · 단청 (Dancheong) — Lessons 1–4
한국 사찰 단청 문양을 그리며 프로그래밍의 기초를 익힙니다.

| 차시 | 개념 | 문화 주제 |
|------|------|----------|
| 1 | 순차적 실행 + 주석 | 단청 채색 순서 |
| 2 | 변수 | 오방색 (다섯 방위색) |
| 3 | 조건문 (if/else) | 방위에 따른 색 선택 |
| 4 | 반복문 (for loop) | 단청 연속 문양 |

### Phase 2 · 한글 (Hangeul) — Lessons 5–8
한글 자음을 코드로 그리고 인터랙티브 앱을 만듭니다.

| 차시 | 개념 | 문화 주제 |
|------|------|----------|
| 5 | 함수 | 한글 자음의 기하학적 구조 |
| 6 | 배열 | 14개 자음 컬렉션 |
| 7 | 객체 | K-POP 곡 데이터 |
| 8 | 통합 (7개 개념) | 한글 단어 플래시카드 앱 |

### Phase 3 · K-POP + AI — Lessons 9–12
AI를 코딩 파트너로 활용하며 자신만의 프로젝트를 완성합니다.

| 차시 | 개념 | 문화 주제 |
|------|------|----------|
| 9 | 바이브 코딩 워크플로우 | K-POP 비주얼라이저 |
| 10 | AI 디버깅 | 단청 × K-POP 융합 |
| 11 | 자유 프로젝트 | 학생이 선택한 주제 |
| 12 | 발표 + 회고 | 완성작 시연 |
