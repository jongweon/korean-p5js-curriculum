Student Setup Guide · Before Your First Class

Get Ready to Code

Follow these steps before Lesson 1 starts. Total setup time: about 15 minutes. You will need a laptop or desktop computer and a stable internet connection.

1
Use Google Chrome
Required browser for all tools in this course
2 min
1

Download and install Chrome

Go to google.com/chrome and download the browser if you don't have it. All tools in this course work best in Chrome.

2

Learn to open the Developer Console

You will use this to read error messages. Press F12 (Windows) or Cmd + Option + J (Mac). Click the Console tab. You will need this from Lesson 10 onward.

Why Chrome? The p5.js Web Editor and Claude.ai both work best in Chrome. Using a different browser may cause unexpected errors during class.
2
Create a p5.js Web Editor Account
Where you will write and run all your code
5 min
1

Go to the p5.js Web Editor

Open a new Chrome tab and go to the address below. This is your coding environment — no installation needed.

2

Click "Sign Up" and create a free account

Use your school email address. Choose a simple username you can remember. You do NOT need a paid plan — the free account has everything you need.

3

Run your first sketch — confirm it works

After signing in, press the ▶ Play button. You should see a grey square appear on the right. If you see it, your editor is working correctly.

4

Enable Auto-save

Click File → Auto-save to turn it on. This prevents losing your work.

Important: Always stay signed in during class. If you close the browser without saving, your sketch will be lost. Use Ctrl+S (Win) or Cmd+S (Mac) to save manually at any time.
3
Access Claude.ai (AI Assistant)
Your AI coding partner for vibe coding exercises
3 min
1

Open Claude.ai in a new tab

Go to the address below. Claude is the AI assistant you will use to generate, debug, and improve p5.js code throughout the course.

2

Create a free account

Sign up with your school email. The free plan gives you enough messages for all 12 lessons. If Claude is unavailable, ChatGPT (chat.openai.com) works as an alternative.

3

Send a test message to confirm it works

Type: Hello! I am going to use you to help me write p5.js code. Please respond in English. — If Claude replies, you are ready.

AI is a tool, not a shortcut. Always read every line of code that AI gives you before pasting it into your editor. You must be able to explain every line at Demo Day (Lesson 12).
4
Open the Curriculum Web App
Your lesson guide, code examples, and AI prompts — all in one place
2 min
1

Open the curriculum app in a new tab

Your teacher will share the link. Bookmark it so you can find it quickly at the start of every lesson.

2

Click on Lesson 1 in the sidebar

Check that you can see the lesson content, the live p5.js canvas preview, the example code, and the AI prompt box at the bottom of the page.

3

Open the AI Prompt Guide tab

Click 🤖 AI Prompt Guide in the sidebar. This is your reference card for writing good AI prompts throughout the course.

5

RECOMMENDED LAYOUT — 3 windows side by side
1

Snap windows side by side (Windows)

Drag each Chrome window to the left or right edge of the screen until it snaps to half-width. Or press Win + ← / Win + →.

2

Use Split View (Mac)

Hold the green full-screen button on a window and drag it to the left or right half of the screen. Then select another window for the other half.

3

Minimum: use browser tabs

If split screen isn't possible, keep all three in the same Chrome window and switch between tabs with Ctrl+Tab (Win) or Cmd+Tab (Mac).

Common Setup Problems
❓ The p5.js editor shows a blank white page or doesn't load.
Try a hard refresh: Ctrl+Shift+R (Win) or Cmd+Shift+R (Mac). If that doesn't work, clear your browser cache: Chrome menu → Settings → Privacy → Clear browsing data.
❓ When I press Play, nothing appears in the preview panel.
Check the Console (F12) for red error messages. The most common cause is a typo in the default code. Try deleting all code and typing: function setup() { createCanvas(400,400); } then pressing Play again.
❓ Claude.ai says I've reached my message limit.
The free plan has a daily message limit. Use ChatGPT (chat.openai.com) as a backup — it understands p5.js equally well. Always paste the same prompt.
❓ I can't log in to my p5.js account.
Use the "Forgot password" link on the login page. If you never received a confirmation email, check your spam folder. As a temporary fix, you can use the editor without logging in — but your sketches won't be saved.
❓ My laptop screen is too small to show three windows.
Use two tabs instead: keep the curriculum app and Claude.ai in separate tabs, and use the p5.js editor as your main window. Press Ctrl+Tab to switch quickly. Alternatively, use an external monitor if available.
